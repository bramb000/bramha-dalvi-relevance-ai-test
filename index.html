I queried the table

prod-all-customer.userbase-table 
SELECT
  COUNT(*) AS total_users,
  AVG(months_subscribed) AS avg_months_subscribed,
  AVG(customer_lifetime_value) AS avg_clv
FROM
  `prod-all-customer.userbase-table`;
My qualitative process

I found this table linked to a Typeform that was has been live since 2 October 2025

typeform-survey.oct-2-2025-feedback-diagnostic-v1 
SELECT
  response_id,
  submitted_at,
  email,
  customer_segment,
  nps_score,
  pain_point,
  feature_request,
  satisfaction_rating,
  completion_time_seconds
FROM
  `typeform-survey.oct-2-2025-feedback-diagnostic-v1`
WHERE
  submitted_at >= '2025-10-02'
ORDER BY
  submitted_at DESC;
